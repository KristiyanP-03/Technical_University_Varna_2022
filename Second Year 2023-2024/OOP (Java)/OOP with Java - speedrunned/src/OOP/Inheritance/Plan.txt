JAVA Е ПЛАТФОРМЕНО НЕЗАВИСИМ ЕЗИК
Компилаторът (javac) преобразува изходния код (.java файл) в bytecode (.class файл). JVM изпълнява байтекода.

JAVA Е ОБЕКТ ОРИЕНТИРАН ЕЗИК
- основните прнципи на обектно ориентирано програмиране:

1 Абстракция
2 Капсулиране
3 Наследяване
4 Полиморфизъм

Java не позволява множествено наследяване на класове

Java няма явни указатели, като тези в езици като C или C++. Вместо това, в Java се използват референции.

Интерпретаторът се наречи JVM (Java Virtual Machine)

Java е компилаторен и интерпретаторен език:
- Компилатора преобразува кода в двуичен байт код      # Създава
- След това кода се run-ва на JVM (интерпретаорено базиран софтуер)    # Изпълнява


За да се стартира Java клас, се създава инстанция на JVM.
